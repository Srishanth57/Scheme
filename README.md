# Government Schemes Filter & Information Portal

This is a [Next.js](https://nextjs.org) project designed to help users easily find and understand various government schemes. The application provides a clean, user-friendly interface with powerful filtering options and detailed information for each scheme.

## Features

* **Advanced Filtering:** Easily search and filter schemes by category, eligibility, and more.
* **Detailed Information:** View comprehensive details for each scheme, including application processes, benefits, and eligibility criteria.
* **Responsive Design:** Fully responsive layout that works seamlessly on desktop and mobile devices.
* **AI Chatbot (In Progress):** An upcoming feature to provide instant answers and assistance through a voice-integrated chatbot.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
